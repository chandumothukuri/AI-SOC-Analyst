# AI SOC Analyst

## Overview

AI SOC Analyst is a Python-based Security Operations Center (SOC) automation project that processes security alerts, assigns risk scores, maps events to MITRE ATT&CK techniques, and generates incident reports.

## Features

* Alert ingestion from JSON files
* Risk scoring based on alert severity
* MITRE ATT&CK mapping
* Automated incident report generation
* Report storage in text format
